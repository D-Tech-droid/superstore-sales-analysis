# superstore-sales-analysis
Excel and SQL analysis of Superstore dataset
Project Overview

This project analyzes a retail Superstore dataset to uncover business insights, sales trends, and profitability performance across regions, categories, and customers.

This project demonstrates practical skills in Excel data analysis and SQL querying.

Tools Used
Microsoft Excel
SQL

Dataset Information
The dataset contains retail sales transactions including:
Orders
Customers
Products
Sales
Profit
Shipping information
Regions and categories

Excel Analysis
Tasks performed in Excel:
Data cleaning and formatting
Pivot Tables creation
Sales vs Profit regional analysis
Dashboard and charts creation

Key Insights:
Identified regions generating high sales but low profit
Compared performance across product categories
Built a visual dashboard for business decision-making

SQL Analysis
SQL queries were used to answer business questions:
Key Queries:
Total sales, orders and profit summary
Top 5 customers by revenue
Top 10 most profitable products
Profit margin by product category
Customers spending above average
